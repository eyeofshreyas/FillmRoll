# Support

## Getting Help

For usage questions, setup issues, or troubleshooting:

- Open a GitHub Issue using the appropriate template.
- Provide enough context to reproduce your problem.

## What to Include

- Operating system
- Python version
- Browser (if UI issue)
- Relevant environment variable setup (without secret values)
- Error logs or screenshots
- Steps already tried

## Security Issues

For security concerns, follow `SECURITY.md` and do not create a public issue.
